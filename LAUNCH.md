## 🔥 Growth Tactics

### Viral Loop Optimization

**Current flow:**
1. User calculates age
2. Sees "12,482 people shared today"
3. Clicks "Challenge a Friend"
4. Gets shareable link with their data encoded
5. Friend clicks → auto-fills compare tab → calculates → shares their own

**Optimize:**
- A/B test social proof numbers (track in GA)
- Add "Share on Twitter" direct button
- Create pre-written tweet templates

### SEO Content Pages

Create these pages for organic traffic:

- `/age-calculator` — Main landing
- `/date-difference-calculator` — Comparison tool
- `/how-many-days-old-am-i` — FAQ format
- `/life-calendar` — Feature spotlight

Each page should:
- Have unique meta description
- Include calculator embed
- Link back to main app
- Target long-tail keywords

### Challenge Mode Virality

**Encourage challenges:**
- Add leaderboard (optional): "Most challenges sent this week"
- Create challenge templates: "I'm X days older than you — prove me wrong"
- Add funny challenge messages based on age gap

## 📈 Metrics to Track

**Week 1 Goals:**
- 1,000 unique visitors
- 200 calculations
- 50 share card downloads
- 10 challenge links created

**Track in GA:**
- `calculate` — single vs compare
- `share_card_open` — which style
- `share_card_download` — conversion rate
- `challenge_generate` — viral coefficient
- `challenge_accept` — loop completion
- `profile_clear` — churn signal

**Key metric:** **Viral Coefficient**
- If 100 users generate 30 challenge links, and 10 friends sign up = 0.1 coefficient
- Goal: Get above 0.3 (30% of users bring 1 new user)

## 🚨 Common Launch Mistakes to Avoid

❌ **Don't:**
- Spam multiple subreddits at once (get banned)
- Launch on Friday (weekend traffic drops)
- Forget to respond to comments (engagement = visibility)
- Ignore negative feedback (iterate fast)

✅ **Do:**
- Engage authentically in communities first
- Post at peak hours (10am-2pm EST)
- Reply to every comment in first 2 hours
- Share behind-the-scenes build story
